# SuperHotSwap
开发初心：旨在做出一款最便捷的IDEA热更新插件，减少用户操作步骤，提供零配置的可视化操作更新。

## 开发环境
- JDK1.8
- IDEA2021.3
- Gradle8.7

## 支持功能
| 支持功能      | 是否支持 |
| ----------- | ----------- |
| MybatisXML热更新      | √       |
| Class热更新   | 进行中        |
| Spring热更新   | 进行中        |
| ...   | ...        |
